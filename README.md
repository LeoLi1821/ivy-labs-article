# 选专业选的不是赛道，是孩子认识自己的方式

Ivy Labs · 教育洞察 — 单页文章网站。

## 预览

本地打开 `index.html` 即可查看，或部署到 GitHub Pages 获得在线访问链接。

## 部署到 GitHub Pages

1. 将本仓库推送到 GitHub。
2. 仓库 **Settings → Pages → Source** 选择 `main` 分支根目录。
3. 等待约 1 分钟，访问：

   ```
   https://<你的用户名>.github.io/<仓库名>/
   ```

## 文件

- `index.html` — 文章页面（自包含，无外部依赖）
